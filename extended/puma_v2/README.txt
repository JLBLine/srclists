##02/02/2015 - J Line
Includes all the image plots (*sourcename*_full.png) and 
SED plots (*sourcename*.png) for all of the extended sources.
The python scripts were used to create the plots. All of the
infomation and logic behind each extended source are logged
in puma-v1_hacklog.txt and puma-v2_hacklog.txt. 
puma-v1_hacklog.txt uses an old naming convention 
unfortunately. The current naming convention is:

EXT%s%s %(catalogue,source_number)

where source_number is the number entry of the catalogue
